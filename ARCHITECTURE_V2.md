# ZeroGravity Business Repository v2

This repository owns public and commercial outputs: the website, portfolio, service definitions, brand, marketing, business operations and project metadata.

Large raw footage does **not** belong in Git. Store it in a managed media library and commit only manifests, rights records, thumbnails, edit notes and small web-ready previews.

## Media lifecycle

Capture → Ingest → Verify ownership/permission → Select → Edit → Review → Export → Portfolio → Website → Archive.

## Operator integration

Business events such as client outreach, quote creation, paid delivery and portfolio publication may be emitted to Zero Command as execution evidence. Financial results remain business metrics; they are not XP by themselves.
